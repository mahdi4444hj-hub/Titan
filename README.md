# TitanFusion

TitanFusion یک پلتفرم امنیتی ویندوزی با نسخه‌های:
- Portable
- Portable Server (بدون پنجره، پایدار 24/7)
- Installer Builder (تبدیل پرتابل به نصب اتومات)

## Features
- Dashboard + API
- HTTPS Dashboard
- API Key + IP Whitelist
- Live Monitoring
- Smart Rate Limit (CPU-based)
- Geo-IP Block
- AI-Based Attack Detection
- Local WAF
- Adaptive Rules
- Daily Reports

## Platforms
- Windows 10+
